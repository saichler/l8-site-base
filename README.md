# l8-site-base

Shared base assets and templates for Layer 8 ecosystem marketing sites.

The reusable website files live under [site-base](./site-base).

This repository is intended to hold the shared non-application website layer:

- brand/site CSS
- nav and reveal JS
- shared images
- reusable page templates for company and component-detail pages

It is intentionally separate from `l8ui`, which remains the application UI library.
